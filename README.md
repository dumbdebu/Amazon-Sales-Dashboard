Amazon Sales Live Interactive Excel Dashboard
Overview

This project is an interactive Excel dashboard designed to analyze Amazon sales data and generate meaningful business insights using advanced Excel features. The dashboard transforms raw e-commerce data into dynamic visual reports for better decision-making and performance tracking.

Features
Interactive Dashboard Interface
Pivot Tables & Pivot Charts
Dynamic Slicers and Filters
KPI Tracking
Sales Trend Analysis
Product Performance Insights
Regional Sales Analysis
Delivery Status Monitoring
Customer Behavior Analysis
Real-Time Data Visualization
Tools & Technologies Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers
Conditional Formatting
Data Cleaning & Transformation
Dashboard Insights

The dashboard helps analyze:

Total Sales Performance
Monthly and Yearly Sales Trends
Top Performing Products
Region-wise Sales Distribution
Order Delivery Performance
Customer Purchase Patterns
Dataset

The dataset contains Amazon sales-related information including:

Order Details
Product Categories
Sales Amount
Delivery Status
Customer Information
Regional Data
Project Objective

To build a professional and interactive business intelligence dashboard in Excel that enables users to explore sales data efficiently and derive actionable insights.

How to Use
Download the Excel dashboard file.
Open in Microsoft Excel.
Enable editing/content if prompted.
Use slicers and filters to interact with the dashboard.
